# TinyLink – Fullstack URL Shortener

## Project Overview

TinyLink is a fullstack web application that enables users to shorten long URLs, generate custom short codes, and view basic click analytics. It consists of a modern frontend for user interaction and a backend API for storing, redirecting, and tracking links. The project demonstrates CRUD operations, RESTful API usage, and integration of a database with an ORM.
